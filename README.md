# Dental Clinic Management System

A Laravel-based web application designed to manage dental clinic operations efficiently. The system helps manage patients, doctors, appointments, medical records, prescriptions, and administrative tasks through a user-friendly dashboard.

## Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- Medical Records Management
- Prescription Management
- User Authentication
- Admin Dashboard
- Database Management
- Responsive User Interface

## Technologies Used
- PHP
- Laravel 12
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript
- Git/GitHub

## Project Structure
The application follows the Laravel MVC architecture:

- Models: Database and business logic
- Controllers: Application logic
- Views: User interface
- Migrations: Database structure management
- Routes: Application endpoints

## Database
The system uses MySQL for storing:

- Patients data
- Doctors information
- Appointments
- Medical records
- Prescriptions
